Project Overview

The Tweet Analyzer is a tool designed to flag tweets as sensitive, political, containing false information, or requiring no flagging, along with providing an explanation for each decision. It retrieves tweets and associated media using the Twikit API, then sends the content to the Gemini API for analysis. The tool operates via a Flask server, which is hosted on an AWS Ubuntu server, the same server where the website is hosted.
